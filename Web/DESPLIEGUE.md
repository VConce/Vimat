# Despliegue

Esta version es un espejo HTML estatico de la web Wix actual. Conserva recursos remotos de Wixstatic, Google Drive y enlaces externos originales.

Para GitHub Pages, publicar el contenido de esta carpeta y usar `index.html` como entrada. El menu se parchea a archivos `.html` locales para evitar rutas de Wix que no existen en GitHub Pages.
